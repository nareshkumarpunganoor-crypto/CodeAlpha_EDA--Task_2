# Netflix EDA Project

## Overview
Performed Exploratory Data Analysis on Netflix Movies and TV Shows dataset to identify trends, patterns, and insights.

## Key Analysis
- Movies vs TV Shows
- Country-wise content analysis
- Yearly content growth
- Ratings distribution
- Genre analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
Netflix Titles Dataset from Kaggle
